# **App Name**: Project Sentinel

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth, supporting multiple roles (admin, manager).
- Project Management: Create, read, update, and delete projects with details like name, location, budget limit, and assigned manager.
- Worker Management: Manage worker profiles, including their skills, contact information, payment type, and base rate.
- Transaction Tracking: Record and categorize project transactions (expense, income, payout advance, payout settlement) with proof image uploads.
- Task Management: Create and manage tasks associated with projects, track completion status, and upload completion photos.
- Attendance Tracking: Record worker attendance, tracking dates, status, and units worked.
- Budget Overview: AI powered tool providing an overview of the budget including possible budget overruns using past transactions

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to evoke trust and reliability.
- Background color: Light gray (#F9FAFB) for a clean and modern interface.
- Accent color: Orange (#EA580C) for calls to action and important information.
- Headline font: 'Space Grotesk' sans-serif, providing a modern tech-centric feel; for body text, use 'Inter' sans-serif.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, minimalist icons from a library like Remix Icon to represent different entities and actions.
- Use subtle transitions and animations to enhance user experience, such as loading spinners and confirmation messages.
